# Perfect Parry

Mobile-friendly precision reaction game built in pure JavaScript using HTML5 Canvas.

## About
Perfect Parry is a minimalist reaction game where players must time shields to parry incoming attacks. The goal is to achieve perfect parries to build combos and score points.

## Features
- Mobile-first controls (touch + tilt ready)
- Smooth 60 FPS canvas rendering
- Particle effects and polish
- Difficulty scaling and progressive attacks

## Controls
- Mouse / Tap: raise shield (hold)
- Space / Touch: parry
- R: reset game

## How To Run
Open `index.html` in a modern browser. For local dev, you can use a simple static server:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Technical Details
- Vanilla JS modules
- Canvas rendering separated from game logic
- devicePixelRatio aware scaling
